## ğŸš€ Getting Started

### Prerequisites
- Node.js 14+ 
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/UCodeCrafter/uttam-resume-app.git
cd uttam-resume-app
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm start
```
The app opens at `http://localhost:3000/uttam-resume-app`
